## Libraries
- [html-webpack-inject-style-plugin](https://github.com/niexia/html-webpack-inject-style-plugin): 📦 Dynamically inject stylesheet `<link />` tags instead of directly including them when generating files, especially helpful for using webpack-rtl-plugin.
- [vue-truncate-list](https://github.com/twheeljs/vue-truncate-list): A flexible Vue 2 & Vue 3 component for truncating lists with custom truncator rendering. Designed for both mobile and desktop platforms.

## Extensions
- [postMessage-catcher](https://github.com/niexia/postMessage-catcher): ⚒️ A Chrome extension that captures the content of postMessage.
- [customize-tab-icon](https://github.com/niexia/customize-tab-icon): ✌️A simple yet powerful Chrome extension that allows you to customize website (browser tab) icons . Built with WXT + React + shadcn/ui + Tailwindcss.
